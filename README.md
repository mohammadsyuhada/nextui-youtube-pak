# YouTube.pak for NextUI (TrimUI Brick)

A YouTube player pak for NextUI on TrimUI Brick. Watch YouTube videos directly on your device.

## Video Player Controls

| Button | Action |
|--------|--------|
| **A** | Play/Pause |
| **B** | Exit/Quit player |
| **LEFT** | Seek backward 5 seconds |
| **RIGHT** | Seek forward 5 seconds |
| **UP** | Seek forward 60 seconds |
| **DOWN** | Seek backward 60 seconds |
| **L2** | Take screenshot (without subtitles) |
| **R2** | Toggle fullscreen |
| **START** | Play/Pause |
| **SELECT** | Play/Pause |
| **MENU** | Exit/Quit player |

## Quick Reference

- **Pause/Resume**: Press B, START, or SELECT
- **Skip forward/back**: Use D-Pad (5 sec with LEFT/RIGHT, 60 sec with UP/DOWN)
- **Exit player**: Press A or MENU button
